# How to Update Your Repository on GitHub

Whenever you make changes to your code (like editing text, adding images, or fixing bugs), follow these steps to save them to GitHub.

## 1. Check Status
See which files have changed:
```powershell
git status
```

## 2. Stage Changes
Prepare all changed files for the update:
```powershell
git add .
```

## 3. Commit Changes
Save the changes with a short description of what you did:
```powershell
git commit -m "Describe your changes here"
```
*Example: `git commit -m "Updated the bio text"`*

## 4. Push to GitHub
Upload your changes to the cloud:
```powershell
git push
```

---
**Quick Cheat Sheet:**
```powershell
git add .
git commit -m "update message"
git push
```
